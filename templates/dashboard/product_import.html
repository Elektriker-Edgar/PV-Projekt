    CSV Import

    Importieren Sie Produkte aus einer CSV-Datei

    Hinweise zum Import
    Wie funktioniert der Import?
    • Format prüfen: Die CSV-Datei muss die korrekte Struktur haben (siehe Vorlage)
    • Trennzeichen: Verwenden Sie Semikolon (;) als Trennzeichen
    • Dezimalzahlen: Verwenden Sie Komma (,) für Dezimalzahlen (z.B. 19,99)
    • ID vorhanden: Wenn die ID-Spalte eine gültige Produkt-ID enthält, wird das bestehende Produkt überschrieben
    • ID fehlt/ungültig: Es wird ein neues Produkt angelegt (ID-Spalte kann leer sein oder eine nicht-existierende ID)
    • Pflichtfelder: Kategorie, SKU (Artikelnummer), Bezeichnung sind erforderlich
    • Kategorien: Kategorien werden automatisch erstellt, wenn sie nicht existieren
    Spaltenformat
    Spalte Beschreibung Erforderlich Beispiel
    ID Produkt-ID (leer lassen für neue Produkte) Optional 1, 2, 3 oder leer
    Kategorie Name der Produktkategorie Pflicht Wechselrichter, Kabel, etc.
    Artikelnummer (SKU) Eindeutige Artikelnummer Pflicht WR-001, KAB-5MM-25
    Bezeichnung Name des Produkts Pflicht Wechselrichter 5kW
    Beschreibung Ausführliche Produktbeschreibung Optional Hocheffizienter 3-phasen...
    Einheit Stück, Meter, Stunde, kWh, kWp, Set, Paket, Pauschal, Prozent Standard: Stück Stück, Meter
    Einkaufspreis (netto) Netto-Einkaufspreis in EUR Standard: 0 450.00
    Verkaufspreis (netto) Netto-Verkaufspreis in EUR Standard: 0 590.00
    MwSt.-Satz (%) 0, 7 oder 19 (Prozent) Standard: 19 19
    Weitere Spalten (Brutto, Margen) werden automatisch berechnet
